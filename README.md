# Sales Dashboard - Power BI

## Overview
Interactive sales dashboard created using Power BI to analyze sales performance.

## Tools Used
- Microsoft Excel
- Power BI

## Data Cleaning
- Checked and handled missing values
- Verified duplicate records
- Standardized formatting
- Converted sales fields into proper numeric formats

## Dashboard Features
- Total Sales KPI
- Total Units Sold KPI
- Sales by Region
- Sales by Product
- Salesperson Performance
- Sales Trends Over Time

## Dataset
Sales transaction dataset containing:
- Order ID
- Order Date
- Product
- Region
- Salesperson
- Units Sold
- Unit Price
- Total Sales
